发表于2020-08-01更新于2023-02-22字数统计1.6k

　　这篇博客主要讲的是如何进行 Hexo&github 博客迁移，此处应用场景为换电脑后进行迁移

　　由于本人电脑有两块固态，所有在空间比较小的固态上装了一个 Ubuntu 系统，安装之后，进行一些开发应用，在安装之后有一些使用的问题想要记录一下，所以决定迁移个人博客到这个系统上。

  2. 安装Nodejs：此处参考本人博客中的Ubuntu安装Nodejs

  3. 安装Hexo：此处参考本人博客中的Ubuntu安装Hexo
